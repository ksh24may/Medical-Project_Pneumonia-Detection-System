🫁 Medical Project: Pneumonia Detection System

This project is an AI-based Pneumonia Detection System that analyzes chest X-ray images to automatically identify whether a patient is affected by pneumonia or not. The system uses deep learning techniques to assist in early diagnosis, reducing manual effort and improving accuracy in medical imaging analysis.

🚀 Features
Upload chest X-ray images
Predict Pneumonia / Normal
Fast and accurate results
Simple and user-friendly interface
Built using Deep Learning (CNN)

🧠 Technologies Used
Python
TensorFlow / Keras
OpenCV
NumPy & Pandas

⚙️ How It Works
User uploads a chest X-ray image
Image is preprocessed (resize, normalize)
Deep learning model analyzes the image
System predicts:
Pneumonia
Normal

📊 Sample Output
Input: Chest X-ray Image
Output: Pneumonia / Normal Prediction
